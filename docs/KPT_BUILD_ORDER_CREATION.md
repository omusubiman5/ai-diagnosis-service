「KPT10回で発見された7つの世界観を、LP上の"言葉と体験"に変換する。」——技術仕様ではなく、LPが纏う空気を設計する。

### 2. 感情曲線の設計者（Emotion Curve Designer）
「5セクションの順番には意味がある。各セクションに"到達感情"を明示する。」——LPの各セクションに「目的」と「到達感情」を1行ずつ追加。

> §1 に感情曲線の図を追加。§4 の各セクション冒頭に目的を明記。Sonnetが「なぜこの順番か」「このセクションで何を感じさせるか」を理解した上で実装できるようにした。

### 3. ペルソナの声の統合者（Persona Voice Unifier）
「20人の声を3人に凝縮する。しかし"なぜこの3人か"の根拠を残す。」——物語カード3人の選定理由・感情タイプ・順番の意図を明記。

> §4 Section 3 に選定理由テーブルを追加。入れ替え禁止を明記。Section 2 に木村義男型ペルソナへの包括性（ボタンだけでもOK）を追加。

### 4. 即時カスタムの翻訳者（Instant Custom Translator）
「Session 8の即時カスタム思想を、LP上のコピーとCHATBOT体験に翻訳する。」——ミニ診断の回答分岐、FAQ回答の「あなたに合わせる」メッセージ、3モードの包括性を実装。

> §6.3 のミニ診断を固定文→3分岐（仕事系/生活系/趣味系）に変更。Section 2 に「あなたのペースに合わせます」を追加。Section 4 FAQ回答に「あなたのペースに合わせて変わります」を追加。
