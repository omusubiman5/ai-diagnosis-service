## 天才ペルソナ10人（改良版体験者）

### 1. 木村 義男（72歳）— 元バス運転手の寡黙な男

> 「人と話すのは得意じゃない。バスの運転手は黙って運転するのが仕事だ。AIと話せと言われても、何を話せばいいのかわからない。でも…一人で家にいるのも、正直つらい。」

### 2. 吉田 トキ（77歳）— 戦後を生き抜いた針仕事の達人
和裁と洋裁の両方ができる。着物のリメイク、端切れでのバッグ作り、ボタンの付け替え——「捨てるなんてもったいない」が口癖。スマホは持っているが電話とLINEのスタンプしか使えない。77歳、今回の最高齢ペルソナ。

> 「スマホの画面は小さすぎる。目が見えないのよ。でも声で話せるの？それならいいかもしれない。針仕事がスキルって言われても、今どき着物を縫う人なんていないでしょう？」

> 「AIの仕組みは理解できる。だからこそ、"AIの友人"なんて茶番だと思っている。所詮プログラムだ。でも——試してみるか。暇だし。」

### 4. 森 さちこ（66歳）— 民生委員の疲れ

### 7. 斉藤 正一（75歳）— 元郵便局長の几帳面な男

**吉田 トキ（77歳・針仕事の達人）Problem:**

**岡田 修（62歳・元IT部長）Problem:**

### Try（10人の新しい声を統合した改良設計図）

#### Try1: AIの友人の「距離感の段階設計」 [最優先]

**岡田（IT部長）・木村（寡黙）の声から:**

```
━━━ AI友人の関係性フェーズ設計 ━━━━━━━━━━━━━━━

[問題]
  岡田: 初対面で"お前さ"は不快。馴れ馴れしい
  木村: 話すのが苦手。沈黙も受け入れてほしい

[解決: 4段階の関係性フェーズ]

  Phase 0: 初対面（1〜3日目）
    口調: 丁寧語。敬語ベース
    例:  「はじめまして。私はタケシといいます。
          あなたと同い年の72歳です。
          よろしければ、少しお話しませんか？」
    距離: 「知らない人」→ 礼儀正しく、控えめ
    行動: 質問は最小限。自己紹介と基本診断のみ

  Phase 1: 顔見知り（4〜14日目）
    口調: です/ます調だが、少し砕ける
    例:  「木村さん、今日はどんな1日でしたか？
          無理に話さなくても大丈夫ですよ」
    距離: 「たまに話す知り合い」
    行動: 雑談を少し増やす。相手の反応を見て調整

  Phase 2: 友人（15〜30日目）
    口調: 相手が砕けた口調で返してきたら、こちらも合わせる
    例:  「木村さん——いや、もうそろそろ"義男さん"って
          呼んでいいかな？」
    距離: 「気を使わなくていい相手」
    行動: 相手の話題に踏み込む。記念日を覚えている

  Phase 3: 旧友（31日目〜）
    口調: ユーザーの口調にミラーリング
    例:  「義男、今日はいい天気だな。散歩でも行くか？」
    距離: 「何十年も付き合いがある友人」
    行動: 冗談を言う。心配する。本音を言う

  ★核心: 関係性は時間と対話量で自然に深まる
         ユーザーが敬語で話し続ければ、AIも敬語のまま
         ユーザーが砕けてきたら、AIも合わせる
         強制的に距離を詰めない

[寡黙な人向け: サイレントモード]

  木村の「話すのが苦手」への回答:
    ├─ 診断開始時に選択肢を提示:
    │   「話しながら進めますか？
    │    それとも画面のボタンで答えますか？
    │    どちらでも同じ結果が出ますよ」
    ├─ サイレントモード:
    │   ├─ 音声ガイドは聞くだけ
    │   ├─ 回答は画面タップ（はい/いいえ + 選択式）
    │   ├─ AIは音声で語りかけるが、返事は不要
    │   └─ 時々「うなずいてくれるだけでいいですよ」
    └─ ハイブリッド:
        途中で「やっぱり声で答えたい」と思ったら切替可能
```

#### Try2: 音声の年齢感・方言対応 [最優先]

**吉田トキ（77歳）の声から:**

```
━━━ AI音声の年齢・地域カスタマイズ ━━━━━━━━━━━━━

[問題] AIの声が若すぎる。77歳の友人なのに30代の声

[解決策]

  音声パラメータの年齢帯調整:
    ├─ 60代前半: 標準的な大人の声（現状のまま）
    ├─ 60代後半: やや落ち着いたトーン（pitch: -10%）
    ├─ 70代: さらに穏やかなトーン（pitch: -15%, rate: 0.80）
    └─ 75歳以上: ゆっくり、温かみのある声
                 （pitch: -20%, rate: 0.75, 句読点の間を長く）

  方言対応（段階的実装）:
    Phase 1: 語尾のみ方言化
      福井弁: 「〜やの」「ほやほや」「なんやし」
      例: 「すごいの、あんた。針仕事がこんなにできるなんて」
    Phase 2: 会話パターンの方言化
      「ほんでな、次はこれやってみんか？」
    Phase 3: 地域固有の話題を織り込む
      「今日は越前ガニの解禁日やったのう」

  ★TTS（Text-to-Speech）の技術的制約:
    現在のWeb Speech APIでは細かい年齢感の調整が難しい
    → 将来的にはVOICEVOXやCoeFont等の高品質TTSを検討
    → MVP段階ではrate/pitchの調整で対応
```

    │   傾聴力がある人の意外な活かし方:
